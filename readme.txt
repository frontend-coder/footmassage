Лендинг Пейдж HTML5 шаблон Footmassage


Микро-подписная страница на инфо-продукт Footmassage

Данная микро-подписная страница выполнена для тестового задания: сбор подписной базы по теме массаж, в замен предлагается «Бесплатное руководство по массажу ног от Владимира Онищенко».

Конечно же, продукт и автор выдуман для придания большей наглядности и завершенности самой страницы, текст выдуман, частично использован «текст-рыба».

Прототип — реально существующие странички в интернете, которые верстаются не вручную, а делаются с помощью конструкторов подписных страниц Wordpress. Я же просто сверстал свой вариант вручную. При необходимости минимизации бюджета на разработку, удобства заказчика могу использовать Visual Composer или другие конструкторы посадочных страниц.
Большую часть времени на ее создание потратил на создание 3-D cover, их научился генерировать буквально неделю назад, что создало дополнительные трудности во время создания этой подписной странички. Их создавал с помощью программы Photoshop. Понемногу ее осваиваю, но навыков катастрофически не хватает, чтобы реализовать свои дизайнерские фантазии.
